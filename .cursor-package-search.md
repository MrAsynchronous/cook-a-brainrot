# NevermoreEngine Package Search Quick Reference

This file helps quickly find packages when asked "does a package for X exist?"

The NevermoreEngine packages are located in the `NevermoreEngine/` subdirectory.

## Quick Search Commands

### Search by functionality/keyword
```bash
# Search package descriptions
grep -r -i "keyword" NevermoreEngine/src/*/package.json | cut -d: -f1 | sort -u

# Search README files
grep -r -i "keyword" NevermoreEngine/src/*/README.md | cut -d: -f1 | sort -u

# Search both at once
grep -r -i "keyword" NevermoreEngine/src/*/package.json NevermoreEngine/src/*/README.md | cut -d: -f1 | sort -u
```

### List all package directories
```bash
ls -d NevermoreEngine/src/*/ | xargs -n1 basename
```

### Get package info for a specific package
```bash
# View README
cat NevermoreEngine/src/<packagename>/README.md

# View package.json
cat NevermoreEngine/src/<packagename>/package.json

# Get just the description
grep '"description"' NevermoreEngine/src/<packagename>/package.json
```

## Common Package Categories

### Core Utilities
- `maid` - Cleanup connections
- `signal` - Signal implementation
- `promise` - Promise implementation
- `binder` - Bind classes to Roblox objects
- `rx` - Reactive programming

### Animation/Tweening
- `spring` - Spring animations
- `acceltween` - AccelTween implementation
- `timedtween` - Timed tweening

### UI/Guis
- `blend` - Declarative UI framework
- `basicpane` - Base UI object
- `qgui` - GUI utilities
- `scrollingframe` - Scrolling frame utilities

### Services
- `datastore` - DataStore wrapper
- `camera` - Camera system
- `remoting` - Remote communication

### Utilities
- `string` - String utilities
- `table` - Table utilities
- `math` - Math utilities
- `geometryutils` - 3D geometry utilities

## Response Format

When a package is found, provide:
1. **Package Name**: `<packagename>` (directory name)
2. **NPM Package**: `@quenty/<packagename>`
3. **Installation**: `npm install @quenty/<packagename> --save`
4. **Description**: Brief description from README or package.json
5. **Docs**: `https://quenty.github.io/NevermoreEngine/api/<PackageName>`

## Example Search Workflow

User: "Does a package for handling animations exist?"

1. Run: `grep -r -i "animation" NevermoreEngine/src/*/package.json | head -20`
2. Results show: `animations`, `animationprovider`, `animationgroup`, `animationtrackutils`
3. Check descriptions in each README.md
4. Respond with relevant packages and installation info
